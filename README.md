# Student Management System (Java)

A simple and clean console-based Student Management System built using Java OOP and a modular, multi-file structure. Supports adding, updating, deleting, searching, and listing students with CSV-based data storage.

# Features

  - Add students
  - Update student details
  - Delete by ID
  - Search by name
  - List all students
  - Auto-save to students.csv

# Technologies Used

  - Java (JDK 8+)
  - OOP (Classes, Objects, Encapsulation)
  - File Handling (CSV)
